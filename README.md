# Election-Headquarters-Management-Back-End
Back-End of manage the electoral headquarters of a representative project.

Description :

This is a back-end of manage the electoral headquarters of a representative project. It is created with Node.js. Its database is MySQL. in the admin panel, you can add, delete and update members.
